  2.Jpa, 
  3.Hibernate,

    (B) Many to One Mapping Uni and Bi Directional.
    (C) Many to Many Mapping Uni and Bi Directional.
  4.Liquibase, 
  5.JUnit Tests, 
  6.Hibernate Validations etc.
